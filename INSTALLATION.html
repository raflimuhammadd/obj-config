üíª System Requirements

Minimum Requirements

Component Requirement
OS Windows 10+, macOS 10.15+, Ubuntu 18.04+
Python 3.8 atau lebih tinggi
RAM 4 GB (8 GB recommended)
Storage 2 GB free space
CPU Intel i5 atau equivalent
GPU Optional (CUDA-compatible untuk performa lebih baik)

Recommended Requirements

Component Specification
RAM 16 GB
GPU NVIDIA GPU dengan CUDA support
Storage SSD dengan 10 GB free space

Software Dependencies

Python >= 3.8
pip >= 21.0
git >= 2.0

üöÄ Installation Steps

Step 1: Clone Repository

# Clone dari GitHub
git clone https://github.com/raflimuhammadd/object-detection.git

# Masuk ke directory
cd object-detection

Step 2: Python Virtual Environment (Recommended) 

# Buat virtual environment
python -m venv venv

# Activate virtual environment
# Untuk macOS/Linux:
source venv/bin/activate

# Untuk Windows:
venv\Scripts\activate

Step 3: Install Python Dependencies

Option A: Menggunakan requirements.txt (jika ada) 

cd python-script
pip install -r requirements.txt

Option B: Manual Installation

# Core dependencies
pip install streamlit==1.30.0
pip install ultralytics==8.0.0
pip install opencv-python==4.8.0
pip install pillow==10.0.0
pip install python-dotenv==1.0.0
pip install requests==2.31.0

# Optional: untuk GPU support
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118

Verifikasi Instalasi

# Cek versi yang terinstall
pip list | grep -E "streamlit|ultralytics|opencv|torch"

Expected output:

opencv-python         4.8.0
pillow               10.0.0
python-dotenv         1.0.0
requests              2.31.0
streamlit             1.30.0
ultralytics           8.0.0
torch                 2.0.0  (jika GPU support)

üß† Model Setup

Download YOLOv11 Models

Models sudah tersedia di folder model-pt/: - best-detect.pt - Custom trained model (recommended) - v11.pt - Model version 11 - yolov11s.pt - YOLOv11 small base model

Jika model tidak ada, download dari:

# Download YOLOv11 base model
cd model-pt
wget https://github.com/ultralytics/assets/releases/download/v0.0.0/yolov11s.pt

# Atau gunakan Python
python -c "from ultralytics import YOLO; YOLO('yolov11s.pt')"

Verifikasi Model

# Cek file model ada dan valid
ls -lh model-pt/

# Expected:
# best-detect.pt  (~6 MB)
# v11.pt          (~19 MB)
# yolov11s.pt      (~22 MB)

‚öôÔ∏è Configuration Setup

Step 1: Copy Environment Template

# Dari root directory project
cp .env.example python-script/.env

Step 2: Edit Configuration

cd python-script
nano .env  # atau gunakan text editor favorit Anda

Step 3: Konfigurasi Minimal (Tanpa Telegram)

Edit .env dengan konfigurasi minimal:

# ** DATABASE
DATABASE_PATH=./db/apd_detection.db

# ** LOCATION & CAMERA
LOCATION=Terminal Pelabuhan A
CAMERA_ID=CAM-001
TIMEZONE=Asia/Jakarta

# ** DETECTION SETTINGS
CONFIDENCE_THRESHOLD=0.5
FPS=10

# ** NOTIFICATION SETTINGS (Disable untuk testing)
ENABLE_TELEGRAM=false
ENABLE_EMAIL=false

# ** UI SETTINGS
SHOW_DETECTION_STATS=true

# ** LOGGING
LOG_LEVEL=INFO
LOG_FILE=./logs/apd_detection.log

üì± Telegram Setup (Optional)

Note: Skip bagian ini jika Anda tidak ingin menggunakan notifikasi Telegram.

Untuk setup lengkap, lihat: python-script/development/TELEGRAM_SETUP.md

Quick Setup:

  1. Buat Bot dengan BotFather
1. Buka Telegram, cari @BotFather
2. Kirim: /newbot
3. Ikuti instruksi, dapatkan TOKEN
  1. Dapatkan Chat ID
1. Personal Chat: 
   - Cari @userinfobot
   - Kirim pesan, copy ID Anda

2. Group Chat:
   - Add @getidsbot ke group
   - Copy group ID (dimulai dengan -)
  1. Update .env
ENABLE_TELEGRAM=true
TELEGRAM_BOT_TOKEN="123456789:ABCdefGHI..."
TELEGRAM_CHAT_ID="your_chat_id_here"
  1. Test Connection
cd python-script
python -c "from telegram_notifier import TelegramNotifier; import config; tn = TelegramNotifier(config.TELEGRAM_BOT_TOKEN, config.TELEGRAM_CHAT_ID); print(tn.test_connection())"

üîß Troubleshooting

Problem: Module not found

# Error: ModuleNotFoundError: No module named 'streamlit'
# Solution:
pip install streamlit ultralytics opencv-python pillow python-dotenv requests

Problem: CUDA not available

# Warning: CUDA not available, using CPU
# Solution (Optional untuk GPU):
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118

Problem: Port 8501 already in use

# Error: Port 8501 is in use
# Solution:
streamlit run app-new.py --server.port 8502

Problem: Database permission denied

# Error: Permission denied: 'db/apd_detection.db'
# Solution:
mkdir -p python-script/db
chmod 755 python-script/db

Problem: Model file not found

# Error: Model file not found
# Solution:
# Pastikan file model ada di model-pt/
ls -la model-pt/best-detect.pt

# Atau download ulang
cd model-pt
python -c "from ultralytics import YOLO; YOLO('yolov11s.pt')"
