ğŸ¦º APD Detection System

Sistem Deteksi APD (Alat Pelindung Diri) Real-Time berbasis Computer Vision

Sistem monitoring keselamatan kerja yang menggunakan YOLOv11 untuk mendeteksi penggunaan Alat Pelindung Diri (APD) secara real-time di area pelabuhan. Sistem ini dapat memproses video, gambar, dan webcam stream untuk mengidentifikasi pelanggaran keselamatan dan mengirimkan notifikasi otomatis.

ğŸ¯ Detection Features

  • Real-time Detection: Deteksi APD secara real-time dari video stream
  • Multi-Source Support: Mendukung webcam, video file, dan gambar
  • Multi-Object Tracking: Menggunakan ByteTrack untuk tracking individu
  • Smart Violation Detection: Deteksi pelanggaran dengan severity levels (LOW, MEDIUM, HIGH, CRITICAL)

  • Interactive Dashboard: Dashboard berbasis Streamlit dengan tema light/dark
  • Real-time Statistics: Statistik deteksi dan compliance rate
  • Violation History: Riwayat pelanggaran dengan filter dan export
  • Visual Reports: Grafik dan visualisasi data deteksi

  • Smart Cooldown: Per-person cooldown untuk mencegah spam notifikasi
  • Severity Filtering: Filter notifikasi berdasarkan severity level

ğŸš€ Quick Start

Prerequisites

# Python 3.8 atau lebih tinggi
python --version

# Pip package manager
pip --version

Installation

  1. Clone Repository
git clone https://github.com/raflimuhammadd/object-detection.git
cd object-detection
  1. Setup Environment
cd python-script
pip install -r requirements.txt  # Jika ada
# atau install manual:
pip install streamlit ultralytics opencv-python pillow python-dotenv requests
  1. Configure Environment
# Copy .env example ke .env
cp ../.env.example .env

# Edit .env dengan konfigurasi Anda
nano .env  # atau text editor lainnya
  1. Setup Telegram Bot (Opsional)
# Ikuti panduan di development/TELEGRAM_SETUP.md
# atau set ENABLE_TELEGRAM=false di .env
  1. Run Application
streamlit run app-new.py

Aplikasi akan berjalan di http://localhost:8501

ğŸ”§ Configuration Highlights

Konfigurasi utama di .env:

# Detection Settings
